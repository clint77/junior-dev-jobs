# JuniorDev Jobs

Jobs for junior developers

### Clone the repo

`git clone https://lukemesiti@bitbucket.org/juniordevio/junior-dev-jobs.git`

### Change into folder

`cd junior-dev-jobs`

### Install

After cloning the git repo:

`npm install`

### Build

`npm run webpack`

### Build - development

`npm run webpack-watch`

### Test 

`npm run test`

### Run

`npm start`

Browse to localhost:3000 to view
